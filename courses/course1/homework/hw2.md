# Homework 2: Prompt Engineering Lab

**GENAI 101 — Foundations of Generative AI**
**Due:** End of Week 5 | **Points:** 50 (5% of final grade)

---

## Learning Objectives

- Apply fundamental prompt engineering techniques (role-setting, specificity, formatting)
- Practice advanced prompting methods (chain-of-thought, few-shot, system prompts)
- Develop iterative refinement skills
- Document and analyze the effect of prompt modifications on output quality

## Instructions

### Part 1: Basic Prompt Engineering (15 points)

Choose a topic you know well (your major, a hobby, a job skill). For that topic, create **three versions** of a prompt asking the AI to explain a concept:

1. **Vague prompt** — A casual, unstructured request (1 sentence)
2. **Structured prompt** — Include role, audience, format, and length constraints
3. **Expert prompt** — Add tone, examples of desired output style, and specific exclusions

For each, submit the prompt and the AI's response. Then write 200 words comparing how the outputs differ and why.

### Part 2: Advanced Techniques (20 points)

Complete all four exercises below using any AI model:

**Exercise A — Chain-of-Thought:**
Ask the AI to solve this problem: *"A store sells notebooks for $3 each and pens for $1.50 each. Maria spent exactly $21 and bought more pens than notebooks. How many of each did she buy?"*

First, ask without chain-of-thought. Then, add "Think step by step" or equivalent. Compare the two responses. Which was more accurate? Why does chain-of-thought help?

**Exercise B — Few-Shot Learning:**
Create a prompt that teaches the AI a custom format by providing 2-3 examples, then asks it to generate new content in that format. Suggestions:
- A unique review format
- A custom data entry style
- A specific email template

Submit your few-shot prompt and the output. Explain how the examples shaped the result.

**Exercise C — System Prompts:**
If your AI tool supports system prompts (or "custom instructions"), create a system prompt that makes the AI behave as a specific professional (e.g., a patent attorney, a nutritionist, a game designer). Then ask it 3 questions in that role. Evaluate how well it maintained the persona.

**Exercise D — Iterative Refinement:**
Start with a simple prompt and refine it through **at least 5 iterations** to improve the output. Document each version of the prompt and what you changed. This should show a clear progression from generic to highly specific output.

### Part 3: Prompt Patterns Cheat Sheet (10 points)

Create a one-page reference guide of prompt engineering patterns you've learned. Include:
- Pattern name
- When to use it
- Example prompt template
- Tips or pitfalls

Aim for at least 6 patterns.

### Part 4: Reflection (5 points)

In 200–300 words, discuss:
- Which technique had the biggest impact on output quality?
- When might advanced prompting NOT be worth the effort?
- How do you see yourself using these skills outside this course?
