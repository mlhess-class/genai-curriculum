# HW1 Answer Key: AI First Impressions

**For TA Use Only**

---

## Part 1: Setup (5 points)

- **Full credit:** Screenshots of 3 platforms with model versions noted
- **Common issue:** Students forget to note the model version (e.g., GPT-4o vs GPT-3.5). Deduct 1 point if version not recorded.

## Part 2: Identical Prompt Testing (25 points, 5 per task)

### Task 1 — Factual Question (5 pts)
**Model answer qualities to look for:**
- Correct definitions: ML is a subset of AI that learns from data; DL is a subset of ML using neural networks with multiple layers
- Should be exactly 3 sentences as requested
- **Common errors:** Students sometimes don't use the exact prompt as specified. Deduct 2 points if prompt was modified between models. ChatGPT and Claude typically get this right; watch for Gemini occasionally adding extra sentences.

### Task 2 — Creative Writing (5 pts)
- Check that all 3 responses are included in full
- No single "correct" answer — looking for documentation, not quality judgment
- **Grading note:** 3 pts for complete responses, 2 pts for noting word count adherence

### Task 3 — Analysis (5 pts)
- All models should produce a table; note if any didn't follow the format instruction
- Common: ChatGPT tends to produce clean markdown tables; Claude sometimes gives bullet lists instead
- **Watch for:** Students only including partial responses

### Task 4 — Code (5 pts)
- Correct palindrome function should: handle case insensitivity, strip spaces (optionally), return boolean
- **Common model behaviors:** ChatGPT usually provides clean commented code; Claude tends to add extensive explanation around the code; Gemini may provide multiple solutions
- Students don't need to verify the code runs, but bonus acknowledgment if they do

### Task 5 — Reasoning (5 pts)
- **Correct answer:** 5 cows and 10 chickens (5×4 + 10×2 = 40 legs, 5+10 = 15 animals)
- All modern models should get this right with step-by-step; note which models showed work vs. jumped to answer
- **Common mistake to watch for:** Students accepting wrong AI answers without checking the math

## Part 3: Comparative Analysis (15 points)

**Strong answers include:**
- Specific examples from the actual responses (not general claims like "ChatGPT was better")
- Nuanced observations (e.g., "Claude gave more detailed explanations but ChatGPT followed the format constraint better")
- Acknowledgment that "best" depends on the task type
- Mention of tone/style differences, not just accuracy

**Red flags:**
- Analysis that could have been written without actually doing the exercise
- Identical or near-identical analysis language to what an AI would produce (check for AI-generated analysis about AI — ironic but common)
- Only 1-2 sentences per question

**Grading guide:**
- 13-15: Specific, insightful, evidence-based
- 10-12: Solid but some general statements
- 7-9: Mostly general, few specifics
- <7: Incomplete or clearly didn't do the comparison

## Part 4: Reflection (5 points)

- **Full credit:** Personal, specific, shows genuine engagement with the exercise
- **Deduct 2 pts:** Generic reflection that could apply to anyone
- **Deduct 3 pts:** Less than 150 words or no genuine reflection
- **Common mistake:** Students treat this as a summary of Part 3 rather than personal reflection

## General Notes

- **AI disclosure:** This assignment explicitly asks students to use AI, but the analysis and reflection must be their own writing. If the analysis reads like AI-generated text, discuss with instructor.
- **Late penalty:** 10% per day, max 3 days
- **Total time estimate:** 3-4 hours
