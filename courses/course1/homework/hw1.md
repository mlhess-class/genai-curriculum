# Homework 1: AI First Impressions — Model Comparison

**GENAI 101 — Foundations of Generative AI**
**Due:** End of Week 3 | **Points:** 50 (5% of final grade)

---

## Learning Objectives

- Gain hands-on experience with multiple generative AI platforms
- Develop the ability to critically compare AI model outputs
- Begin building intuition for model strengths and weaknesses
- Practice structured observation and documentation

## Instructions

### Part 1: Setup (5 points)

Create free accounts on **three** AI platforms. Choose from:
- ChatGPT (OpenAI) — chat.openai.com
- Claude (Anthropic) — claude.ai
- Gemini (Google) — gemini.google.com
- Other (with instructor approval)

Take a screenshot of each interface. Note the date and model version shown.

### Part 2: Identical Prompt Testing (25 points)

Submit the **exact same prompt** to all three models for each of the following 5 tasks. Copy the full response from each model.

**Task 1 — Factual Question:**
> "Explain the difference between machine learning and deep learning in exactly 3 sentences."

**Task 2 — Creative Writing:**
> "Write a 100-word short story about a robot discovering music for the first time."

**Task 3 — Analysis:**
> "What are the top 3 pros and top 3 cons of remote work? Present them in a table."

**Task 4 — Code:**
> "Write a Python function that checks if a string is a palindrome. Include comments explaining each line."

**Task 5 — Reasoning:**
> "A farmer has 15 animals — some chickens and some cows. Together they have 40 legs. How many of each animal does the farmer have? Show your work step by step."

### Part 3: Comparative Analysis (15 points)

Write a 500–750 word analysis addressing:
1. Which model was most accurate? Give specific examples.
2. Which model had the best writing style/tone? Why?
3. Did any model make errors or produce questionable content? Describe.
4. Which model would you recommend for each type of task? Justify.
5. What surprised you most about the differences (or similarities)?

### Part 4: Reflection (5 points)

In 150–250 words, reflect on:
- Your prior experience with AI (if any)
- How this exercise changed your perception of AI capabilities
- One question this exercise raised for you that you'd like to explore

## Submission Format

- Single PDF or Word document
- Include all screenshots, prompts, full responses, analysis, and reflection
- Label each section clearly
- Submit via the course LMS by 11:59 PM on the due date
