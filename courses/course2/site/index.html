Course Description

This course bridges the gap between understanding AI and putting it to work. You'll learn workflow-driven thinking, master the open-source n8n automation platform, connect dozens of services through APIs, and build AI-powered pipelines that run autonomously.

By the end of the semester you will have designed, built, tested, and deployed a production-grade automation workflow that integrates at least one AI model, multiple data sources, and robust error handling.

Learning Outcomes

  • Design workflows using inputs → process → output thinking
  • Build and debug n8n workflows with 20+ node types
  • Integrate OpenAI, Anthropic, and local LLMs into automations
  • Work with REST APIs, webhooks, and authentication flows
  • Transform and route data using JSON, expressions, and functions
  • Handle errors, retries, and monitoring in production
  • Deploy self-hosted n8n with security best practices

Weekly Schedule

Week 1

Introduction to Workflow Thinking

Inputs, processes, outputs. Mental models for automation. Identifying repetitive tasks. Mapping manual processes to automated flows. What makes a good automation candidate.

Week 2

The Automation Landscape

Survey of platforms: Zapier, Make (Integromat), n8n, Power Automate, Tray.io. Trade-offs: cost, flexibility, self-hosting, vendor lock-in. Why we focus on n8n — open source, code-optional, AI-native.

Week 3

n8n Fundamentals

Installation (Docker, npm, desktop). The canvas UI — nodes, connections, executions. Building your first workflow: trigger → action → output. Manual vs. production execution. Credentials vault.

Week 4

Working with APIs

REST basics: GET, POST, PUT, DELETE. Headers, query params, request bodies. Authentication: API keys, OAuth2, Bearer tokens. Reading API docs. Postman/Hoppscotch for testing. Rate limits and pagination.

Week 5

n8n Core Nodes Deep Dive

HTTP Request node, Webhook trigger, Function node (JavaScript), IF/Switch routing, Merge node, Set node, Date & Time. Expressions and the $json/$node syntax.

Week 6

AI Nodes in n8n

OpenAI node, Anthropic node, Ollama (local models). Chat completions, embeddings, image generation. Prompt engineering within workflows. Chaining AI calls. Structured output with JSON mode.

Week 7

Data Transformation

JSON parsing and construction. Mapping fields between services. Splitting arrays into items, aggregating items back. Code node for custom transforms. Binary data handling (files, images, PDFs).

Week 8

MIDTERM PROJECT

Build an automated content pipeline: ingest source material → AI summarization/transformation → format for multiple channels → publish or queue. In-class work session + presentations.

Week 9

Error Handling, Retries & Monitoring

Error workflows, try/catch patterns, retry on failure. Dead-letter queues. Execution logs and debugging. Setting up alerts (email, Slack) on failure. Health-check workflows.

Week 10

Database Integration

Postgres node: read, write, upsert. Google Sheets as a lightweight DB. Airtable for structured data. Choosing the right storage. SQL basics for automation engineers.

Week 11

Advanced Workflow Patterns

Sub-workflows and the Execute Workflow node. Looping over large datasets. Parallel execution and wait nodes. Cron and scheduling. Workflow versioning and import/export.

Week 12

Building AI Agents in n8n

Agent node, tool-use patterns, memory (window buffer, vector store). RAG workflows: embed → store → retrieve → generate. Connecting to Pinecone, Qdrant, Supabase pgvector.

Week 13

Production Deployment

Self-hosting with Docker Compose. Environment variables and secrets. Reverse proxy (Nginx/Caddy). SSL certificates. Queue mode for scaling. Backup strategies. Security hardening.

Week 14

Real-World Case Studies

Marketing automation (lead scoring + nurture), customer support (ticket triage + auto-reply), data pipelines (ETL from APIs to data warehouse). Guest speaker session. Ethical considerations.

Week 15

Final Project Presentations

Live demos of student-built AI-powered automation workflows. Peer review. Course retrospective. What's next: GENAI 301 preview.

Policies

Prerequisites

GENAI 101: Foundations of Generative AI (or equivalent with instructor approval). You should be comfortable prompting LLMs, understand tokens/context windows, and have basic familiarity with JSON.

Required Tools

Laptop with Docker Desktop installed. Free n8n account (n8n.cloud) for early weeks; self-hosted instance for later weeks. GitHub account. Free-tier API keys for OpenAI and/or Anthropic.

Late Work

Assignments lose 10% per day late, up to 3 days. After 3 days, no credit unless pre-arranged. Extensions granted for documented emergencies — communicate early.

AI Use Policy

You are expected to use AI tools — this is an AI course. However, you must understand what your workflows do. "I asked ChatGPT to build it" without comprehension earns zero credit. Document your AI-assisted process.

Academic Integrity

Collaboration is encouraged; copying is not. Each student submits their own workflows. Cite any templates, tutorials, or community workflows you adapt. Plagiarism detection applies to written work.

Accessibility

If you need accommodations, contact the Disability Services office and inform me within the first two weeks. I'll work with you to ensure equitable access to all materials and assessments.
