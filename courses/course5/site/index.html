Course Overview

Description

The capstone course synthesizes everything you've learned across the Generative AI curriculum. Working in teams of 3–4, you will identify a real-world problem, design an AI-powered solution, build a working prototype, and present it at a public showcase. This is your opportunity to demonstrate mastery — from stakeholder analysis and architecture design through implementation, testing, and business justification.

Prerequisites

GENAI 401 — Advanced AI Applications (or equivalent, with instructor approval). Students should be comfortable with prompt engineering, model evaluation, API integration, and responsible AI principles.

Learning Outcomes

  • Identify and scope a real-world problem suitable for an AI solution
  • Design end-to-end system architecture for AI applications
  • Build and iterate on a functional prototype using modern AI tools
  • Conduct user testing and incorporate feedback
  • Develop a business case with cost analysis and ROI projections
  • Present technical work to both expert and non-expert audiences
  • Assemble a professional-quality portfolio

Format

Weekly meetings combine short lectures, team workshops, and milestone check-ins. The majority of work happens outside class in your project teams. Expect 8–12 hours per week of out-of-class work. Office hours and Slack channels available throughout.

Policies

AI Tool Usage

You are expected to use AI tools (LLMs, code assistants, image generators) in your capstone work — this is an AI course. However, you must document all AI tool usage and demonstrate understanding of AI-generated outputs. Passing off AI output without comprehension or attribution is an academic integrity violation.

Team Work

Peer evaluations are conducted at Weeks 6, 11, and 15. Persistent issues with contribution equity may result in individual grade adjustments of up to ±15%. Conflicts should be raised with the instructor early.

Late Policy

Milestones are team commitments. Late submissions lose 10% per day, up to 3 days. After 3 days, the milestone receives a zero. Extensions require 48-hour advance notice with documented justification.

Academic Integrity

All work must be original to your team. External code and datasets must be properly attributed. Fabricating user test data or results is grounds for course failure. When in doubt, ask.
