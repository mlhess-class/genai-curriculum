GENAI 301: Building with LangChain

Design and deploy production-grade LLM applications using LangChain, RAG pipelines, agents, memory systems, and LangGraph. Hands-on, code-first.

📅 15 Weeks 🎓 3 Credits 💻 Python + LangChain v0.2+ 📋 Prereq: GENAI 201

Weekly Schedule

LangChain Architecture Overview

Working with LLMs
OpenAI, Anthropic, HuggingFace, Ollama — provider integrations, streaming, fallbacks

Prompt Templates & Output Parsers
ChatPromptTemplate, few-shot, PydanticOutputParser, StrOutputParser, structured output

Chains
Sequential chains, router chains, transform chains, RunnablePassthrough, RunnableParallel

Introduction to RAG

Vector Databases

Agents & Tools
ReAct pattern, function calling, custom tools, tool selection, agent executors
WK 11
Memory Systems
Buffer, summary, conversation, entity memory — persistence patterns and token management
WK 12
LangGraph
Stateful multi-step workflows, nodes, edges, conditional routing, checkpointing
WK 13
Evaluation
LangSmith tracing, custom metrics, automated testing, dataset management
WK 14
Production Patterns
Streaming, caching, cost management, error handling, deployment strategies
WK 15
Final Project Presentations

Prerequisites

  • GENAI 201 — AI-Powered Workflows & Automation (or equivalent)
  • Intermediate Python (functions, classes, decorators, async basics)
  • Familiarity with REST APIs and JSON
  • Basic understanding of how LLMs work (tokenization, attention, generation)
  • Git & command line proficiency
  • Recommended: experience with at least one LLM API (OpenAI, Anthropic, etc.)

Homework Assignments

HW 1
LangChain Foundations & LLM Integration
Due: End of Week 3. Set up a LangChain development environment. Invoke LLMs using LCEL (LangChain Expression Language).
HW 2
Prompt Engineering & Output Parsing
Due: End of Week 5. Master ChatPromptTemplate and few-shot prompting. Build robust output parsers with Pydantic.
HW 3
RAG Pipeline Construction
Due: End of Week 7. Build a complete RAG pipeline from document loading to retrieval. Compare vector stores: ChromaDB, FAISS.
HW 4
Agents & Tools
Due: End of Week 10. Build custom tools for LangChain agents. Implement ReAct agents with function calling.
HW 5
Memory Systems & LangGraph
Due: End of Week 12. Implement and compare LangChain memory systems. Build stateful workflows with LangGraph.
HW 6
Evaluation & Production Patterns
Due: End of Week 14. Evaluate LLM applications with LangSmith. Implement production patterns: streaming, caching, cost management.

Course Policies

AI Usage Policy
You are encouraged to use AI tools throughout this course — it's what we're building with. However, you must always disclose when and how AI was used in any submission. Undisclosed AI use is an academic integrity violation.
Late Work
Assignments lose 10% per day late, up to 3 days. After 3 days, no credit unless a prior extension was arranged. Extensions must be requested at least 24 hours before the deadline.
Attendance
Attendance is expected for all lectures and labs. More than 3 unexcused absences will result in a participation grade penalty. If you must miss class, notify the instructor in advance.
Accessibility
If you need accommodations, please contact the Office of Disability Services and share your letter with the instructor within the first two weeks. We are committed to making this course accessible to everyone.
