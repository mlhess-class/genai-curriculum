Prerequisites

  • GENAI 201 — Foundations of Generative AI (or equivalent)
  • Intermediate Python (functions, classes, decorators, async basics)
  • Familiarity with REST APIs and JSON
  • Basic understanding of how LLMs work (tokenization, attention, generation)
  • Git & command line proficiency
  • Recommended: experience with at least one LLM API (OpenAI, Anthropic, etc.)

Required Tools & Setup

  • Python 3.11+ with pip/uv
  • langchain, langchain-openai, langchain-anthropic, langchain-community
  • chromadb, faiss-cpu, pinecone-client
  • langgraph, langsmith
  • OpenAI API key (required) — $20 credit provided
  • Ollama (local models) — free, recommended
  • VS Code or PyCharm with Python extension
